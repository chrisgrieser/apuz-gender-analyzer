# Name Gender analyzer
Analysis of names by gender.

## Procedure
1. Scrap names from `APuZ`.
2. Get list of names.
3. Parse names into required format.
4. Assign gender to every name. <https://genderize.io/>
5. Calculate percentage of genders.
6. Display results.
